# Forma - AI Powered Decision Intelligence & Forms

![Forma Banner](public/og-image.jpg)

Forma is a cutting-edge platform designed to revolutionize data collection and decision-making. By combining a powerful, no-code form builder with advanced AI analytics, Forma empowers businesses to gather insights and act on them instantly.

## Features

- **Intuitive Form Builder:** Drag-and-drop interface to create stunning forms in minutes.
- **AI-Powered Analytics:** Automatically analyze responses and generate actionable insights.
- **Secure & Scalable:** Built with enterprise-grade security and scalability in mind.
- **Customizable:** Fully brandable forms to match your organization's identity.
- **Real-time Collaboration:** Work together with your team seamlessly.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <YOUR_GIT_URL>
   ```
2. Navigate to the project directory:
   ```bash
   cd quick-data-gather
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Technology Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn-ui
- **Backend/Database:** Supabase
- **Animation:** Framer Motion
